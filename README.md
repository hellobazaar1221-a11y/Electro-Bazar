# Electro Bazar 🛍️

A complete mobile e-commerce app built with Expo React Native.

## Features
- Full product catalog with categories
- Cart & Wishlist
- Orders with tracking
- COD & UPI Payment
- Dark/Light/System theme
- Hindi & English language support
- Push notifications
- Admin panel
- Animated splash screen

## Tech Stack
- Expo (React Native)
- Supabase (Auth + Database)
- Expo Router
- TypeScript
